## Coca-Cola-Sales-Performance-Analytics
## Project Objective
This project focuses on analyzing sales data for the Coca-Cola company, regional performance, and retailer efficiency. The objective is to identify growth trends, evaluate profitability across different markets, and provide actionable insights through data-driven summaries.
