## Coca-Cola-Sales-Performance-Analytics
## Project Objective
This project focuses on analyzing sales data for the Coca-Cola company, regional performance, and retailer efficiency. The objective is to identify growth trends, evaluate profitability across different markets, and provide actionable insights through data-driven summaries.

## Dataset used
- <a href="https://github.com/mohansivakumarmaguluri/Coca-Cola-Sales-Performance-Analytics/blob/main/coco_cola%20sales.xlsx">Dataset</a>

## Questions (KPIs)
Which beverage brand has the highest total sales?
Which retailer is the most profitable in terms of operating profit?
Which region sold the highest number of units?
Which city recorded the highest total revenue?
Which state has the highest average operating margin?
Which beverage brand has the lowest total sales?
Which retailer has the highest average price per unit?
Which month saw the lowest sales in the year 2022?
Which product brand generates the most operating profit?
Which region has the lowest total sales?
